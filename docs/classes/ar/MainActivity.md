# وصف فئة MainActivity

________________________________________
## 1. معلومات عامة
*   **اسم الفئة:** `MainActivity`
*   **النوع:** Activity (نشاط)
*   **الغرض من الفئة:** الشاشة الرئيسية للتطبيق. تعمل كنقطة دخول وقائمة تنقل لعرض عناصر واجهة المستخدم المخصصة المختلفة.
*   **التفاعل:** تطلق Activities أخرى (`DinamicSinus`, `AddCustomActionBar`, `MyToolbar`) وتستدعي فئات مساعدة (`MyToast`, `AlertDialog`).

________________________________________
## 2. المتغيرات (حقول الفئة)
| الاسم | النوع | الغرض | أين يستخدم |
| :--- | :--- | :--- | :--- |
| `context` | `Context` | يحتفظ بمرجع لبيئة النظام. | يمرر إلى Toast و Dialog و Intent. |
| `bToast` | `Button` | زر لعرض Toast مخصص. | `initComponents()`, `onCreate()` |
| `bDialog` | `Button` | زر لفتح مربع حوار معقد. | `initComponents()`, `onCreate()` |
| `bSinus` | `Button` | زر للانتقال إلى رسم الجيب. | `initComponents()`, `onCreate()` |
| `bActionBar` | `Button` | زر للانتقال إلى ActionBar مخصص. | `initComponents()`, `onCreate()` |
| `bToolbar` | `Button` | زر للانتقال إلى Toolbar مخصص. | `initComponents()`, `onCreate()` |

________________________________________
## 3. طرق الفئة (Methods)

### الطريقة: onCreate
*   **النوع:** public
*   **القيمة المرجعة:** void
*   **المعاملات:**
    | الاسم | النوع | الوصف |
    | :--- | :--- | :--- |
    | `savedInstanceState` | `Bundle` | حالة النظام المحفوظة (إن وجدت). |
*   **ماذا تفعل الطريقة:**  
    1. تستدعي طريقة الأب `super.onCreate`.
    2. تضبط تخطيط الواجهة `R.layout.activity_main`.
    3. تستدعي `initComponents()` لربط الأزرار.
    4. تعين مستمعي النقر (`setOnClickListener`) لجميع الأزرار. عند النقر على `bDialog` ، يتم إنشاء وعرض `AlertDialog` معقد مع حقول إدخال وأزرار إجراءات (SMS ، بريد إلكتروني ، إلخ).
*   **متى تستدعى:** عندما ينشئ النظام النشاط.
*   **ما هو المهم فهمه:** تتركز جميع منطق التهيئة هنا. إنشاء مربع حوار داخل هذه الطريقة يجعل الكود ضخمًا؛ كان من الأفضل نقل إنشاء مربع الحوار إلى طريقة منفصلة.

## 8. شرح مبسط
**شرح بكلمات بسيطة:**  
تخيل أن `MainActivity` هي **القائمة الرئيسية في لعبة**. لديك أزرار مثل "ابدأ اللعبة" و "الإعدادات" و "خروج". كل زر يؤدي إلى قسم خاص به. `MainActivity` نفسها لا تلعب اللعبة؛ إنها تساعدك فقط في اختيار الوضع المطلوب.
