# Documentation for Other Classes (English)

---

## Class: CustomToolbar
*   **Type:** Custom View (`LinearLayout`)
*   **Purpose:** Unique toolbar with a logo, title, and a rotating spiral.
*   **Logic:** Uses `RotateAnimation` for the spiral and `PopupMenu` for the menu.
*   **Simple Words:** It's a "smart" header that creates its own icons and even makes one spin.

---

## Class: MyToast
*   **Type:** Utility Class
*   **Purpose:** Shows custom popup notifications with images.
*   **Logic:** Inflates a custom XML layout and sets images based on an integer code.
*   **Simple Words:** Like a greeting card where we choose the envelope and the picture.

---

## Class: DinamicSinus
*   **Type:** Activity
*   **Purpose:** Container for `CustomSinusView`.
*   **Logic:** Programmatically creates and sets the custom view as the content.

---

## Class: AddCustomActionBar
*   **Type:** Activity
*   **Purpose:** Demonstrates a custom ActionBar with a looping video.
*   **Logic:** Replaces the standard ActionBar view with a custom layout containing a `VideoView`.
*   **Simple Words:** Replaces the boring top text with a modern TV screen showing video.

---

## Class: MyToolbar
*   **Type:** Activity
*   **Purpose:** Shows an Activity that includes `CustomToolbar`.
