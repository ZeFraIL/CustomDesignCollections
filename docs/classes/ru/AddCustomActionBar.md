# Описание класса AddCustomActionBar

________________________________________
## 1. Общая информация
*   **Имя класса:** `AddCustomActionBar`
*   **Тип:** Activity
*   **Назначение:** Демонстрация возможности полной замены стандартного ActionBar (верхней панели) на свой макет, включая видео!
*   **Взаимодействие:** Запускается из `MainActivity`.

________________________________________
## 3. Методы класса
### Метод: onCreate
*   **Что делает:** 
    1. Получает доступ к `ActionBar`.
    2. Включает режим кастомного вида (`setDisplayShowCustomEnabled(true)`).
    3. Устанавливает макет `R.layout.custom_action_bar`.
    4. Находит в этом макете `VideoView`.
    5. Загружает видеофайл из ресурсов (`R.raw.mvideo`) и запускает его по кругу.

________________________________________
## 8. Упрощенное объяснение
Обычно вверху приложения просто написано его название. Этот класс делает так, чтобы вверху вместо скучного текста крутилось видео. Это как заменить обычную вывеску магазина на современный ЖК-экран с рекламой.
