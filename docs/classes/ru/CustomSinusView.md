# Описание класса CustomSinusView

________________________________________
## 1. Общая информация
*   **Имя класса:** `CustomSinusView`
*   **Тип:** Custom View (Пользовательский графический компонент)
*   **Назначение:** Отрисовка динамического графика синусоиды. Класс отвечает за математический расчет точек и их визуализацию на холсте (Canvas).
*   **Взаимодействие:** Используется внутри Activity (например, `DinamicSinus`) как обычный визуальный элемент.

________________________________________
## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `paint` | `Paint` | Инструмент для рисования (цвет, толщина линий). | `init()`, `onDraw()` |
| `pointsCount` | `int` | Количество точек в графике (100). | `init()`, `onDraw()` |
| `xPoints` | `float[]` | Массив координат X для графика. | `init()`, `onDraw()` |
| `yPoints` | `float[]` | Массив координат Y для графика. | `init()`, `onDraw()` |
| `currentPoint` | `int` | Счетчик текущей отрисованной точки (для анимации). | `onDraw()` |
| `duration` | `long` | Длительность полной анимации (10 сек). | `onDraw()` |
| `startTime` | `long` | Время начала анимации. | `init()`, `onDraw()` |

________________________________________
## 3. Методы класса

### Метод: init
*   **Тип метода:** private
*   **Возвращаемое значение:** void
*   **Что делает метод:**  
    Настраивает кисть (`paint`): синий цвет, толщина 5, стиль "обводка". Также заранее рассчитывает значения синуса для 100 точек и сохраняет их в массивы.
*   **Когда вызывается:** В конструкторах при создании объекта.

### Метод: onDraw
*   **Тип метода:** protected
*   **Возвращаемое значение:** void
*   **Параметры:** `Canvas canvas` (Холст для рисования).
*   **Что делает метод:**  
    1. Рассчитывает масштаб графика исходя из размеров экрана (`width`, `height`).
    2. Рисует линии между точками от 0 до `currentPoint`.
    3. Вычисляет прогресс времени. Если 10 секунд еще не прошло, увеличивает `currentPoint`.
    4. Вызывает `postInvalidateDelayed(100)`, что заставляет систему перерисовать экран через 0.1 секунды. Это создает эффект анимации.
*   **Когда вызывается:** Автоматически системой при необходимости отрисовки.

________________________________________
## 5. Взаимодействие с интерфейсом (UI)
Класс сам является элементом интерфейса. Он не использует кнопки, а "рисует" себя на экране.

________________________________________
## 7. Общая логика класса
При создании класс вычисляет форму синуса. Затем он начинает "постепенно" показывать эту форму, рисуя по одной точке каждые 100 миллисекунд.

________________________________________
## 8. Упрощенное объяснение
**Объяснение простыми словами:**  
Представьте художника, который рисует картину. Но вместо того, чтобы показать сразу всё, он проводит по одной линии каждые несколько секунд, пока вы на него смотрите. `CustomSinusView` — это такой цифровой художник-математик.

________________________________________
**Совет по коду:**  
В методе `onDraw` создаются локальные переменные `width` и `height`. Это нормально, но расчет `xScale` и `yScale` можно было бы перенести в `onSizeChanged`, чтобы не считать их заново при каждом кадре анимации.
