# Описание класса CustomToolbar

________________________________________
## 1. Общая информация
*   **Имя класса:** `CustomToolbar`
*   **Тип:** Custom View (на базе `LinearLayout`)
*   **Назначение:** Создание уникальной панели инструментов (шапки) приложения. Содержит иконку, заголовок и вращающийся анимированный элемент.
*   **Взаимодействие:** Вставляется в макеты Activity для отображения верхней навигационной панели.

________________________________________
## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `leftImageView` | `ImageView` | Картинка-логотип слева. | `init()` |
| `titleTextView` | `TextView` | Текст заголовка. | `init()` |
| `rotatingSpiral` | `AppCompatImageView` | Иконка спирали, которая вращается. | `init()` |

________________________________________
## 3. Методы класса

### Метод: init
*   **Тип метода:** private
*   **Что делает метод:**  
    1. Устанавливает горизонтальную ориентацию.
    2. Создает "код" для иконки волка, заголовка "ZeFra Toolbar" и спирали.
    3. Создает объект `RotateAnimation` (анимация вращения) на 360 градусов.
    4. Запускает бесконечное вращение для `rotatingSpiral`.
    5. Назначает клик по спирали для вызова меню.
    6. Добавляет все элементы в `LinearLayout` (в себя).

### Метод: showPopupMenu
*   **Тип метода:** private
*   **Что делает метод:**  
    Создает выпадающее меню (`PopupMenu`), берет список пунктов из файла `R.menu.my_menu` и показывает его пользователю.
*   **Когда вызывается:** При нажатии на вращающуюся спираль.

## 8. Упрощенное объяснение
**Объяснение простыми словами:**  
Это "умная" шапка для приложения. Она не просто показывает текст, а сама создает внутри себя картинки и даже заставляет одну из них постоянно крутиться. Это как неоновая вывеска над входом в магазин, которая еще и открывает список услуг при нажатии.
