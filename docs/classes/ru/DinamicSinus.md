# Описание класса DinamicSinus

________________________________________
## 1. Общая информация
*   **Имя класса:** `DinamicSinus`
*   **Тип:** Activity
*   **Назначение:** Экран для отображения динамического графика синуса.
*   **Взаимодействие:** Запускается из `MainActivity`, использует `CustomSinusView`.

________________________________________
## 3. Методы класса
### Метод: onCreate
*   **Что делает:** 
    Создает объект `CustomSinusView` программно (не через XML), задает ему размеры (400x300 плотных пикселей) и устанавливает его как основное содержимое экрана (`setContentView`).

________________________________________
## 8. Упрощенное объяснение
Это просто "рамка" или "комната", внутри которой мы размещаем наш холст с рисующимся графиком.
