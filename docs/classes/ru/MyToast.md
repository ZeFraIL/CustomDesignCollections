# Описание класса MyToast

________________________________________
## 1. Общая информация
*   **Имя класса:** `MyToast`
*   **Тип:** Обычный класс (Utility Class)
*   **Назначение:** Создание и отображение красивых всплывающих уведомлений с картинками.
*   **Взаимодействие:** Вызывается из любой Activity (например, из `MainActivity`).

________________________________________
## 3. Методы класса

### Метод: showToast (статический)
*   **Параметры:**
    | Имя | Тип | Описание |
    | :--- | :--- | :--- |
    | `context` | `Context` | Где показать уведомление. |
    | `message` | `String` | Текст сообщения. |
    | `i` | `int` | Код картинки (1 — приветствие, 2 — прощание). |
*   **Что делает метод:**  
    1. Берет XML-файл `my_toast` и превращает его в Java-объект (`LayoutInflater`).
    2. Находит внутри него текст и картинку.
    3. В зависимости от числа `i` ставит нужную иконку.
    4. Создает стандартный объект `Toast`, но заменяет его обычный вид на наш кастомный.
    5. Показывает результат в центре экрана.

________________________________________
## 8. Упрощенное объяснение
**Объяснение простыми словами:**  
Обычный Toast в Android — это просто серое облачко с текстом. `MyToast` — это как праздничная открытка: мы сами выбираем "конверт" (дизайн), кладем туда картинку и пишем текст, а потом показываем пользователю.
