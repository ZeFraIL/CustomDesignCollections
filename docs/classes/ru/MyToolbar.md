# Описание класса MyToolbar

________________________________________
## 1. Общая информация
*   **Имя класса:** `MyToolbar`
*   **Тип:** Activity
*   **Назначение:** Экран-контейнер для демонстрации `CustomToolbar`.
*   **Взаимодействие:** Запускается из `MainActivity`.

________________________________________
## 3. Методы класса
### Метод: onCreate
*   **Что делает:** 
    Загружает макет `activity_my_toolbar`. В этом XML-файле прописан наш компонент `CustomToolbar`.

________________________________________
## 8. Упрощенное объяснение
Это пустая комната, на стене которой висит наша кастомная панель управления (`CustomToolbar`).
